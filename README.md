        Date: July 2025

Project Description:
---------------------
This project is a responsive single-page frontend UI for "Foodie Hub", a public food ordering platform. It was created as part of a student assignment using only HTML, Bootstrap 5, and custom CSS (no JavaScript).

Features Implemented:
---------------------
1. Responsive Navbar with anchor links (Home, Restaurants, Menu, About, Contact)
2. Hero section with welcome text and CTA
3. Restaurant section with 6 cards (Available/Unavailable badges)
4. Menu section with 6 international dishes and availability badges
5. Order form with customer info and item selection
6. About section with two-column layout
7. Testimonial section with 3 sample customers
8. Achievements section with Bootstrap cards
9. Footer with contact and social media icons

Tools & Technologies Used:
--------------------------
- HTML5
- Bootstrap 5 CDN
- Bootstrap Icons CDN
- Custom CSS (style.css)
- Responsive media queries
